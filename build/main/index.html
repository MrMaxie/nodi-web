Nodi

About creating fast, elegant & modern apps

More than library

Nodi is not bundle of just libraries. It's powered by well documented concepts that allows for programmers write faster and better applications

Flexible

Flexibility is one major rules of Nodi. We won't force anyone to use specific libraries. Instead of that code have to be opened in such way to allows for overriding default behavior

Elegant

Readable code allows not only fix/change code faster, but also develop faster new features as well

Cost meter

It's easy to keep eyes on your call costs to avoid more complex responses, from checking where people need to wait the most

Magicless

This library doesn't contains any magic in itself, only few metaprogramming, well documented tricks. Thanks to that it's easier to keep DRY rule without additional learning

Easy

Nodi does't contains any new syntaxes, any config files, any strict rules for directories structure, any strict rules for software purpose
